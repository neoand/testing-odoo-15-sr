# Análise de Código

Realizar análise profunda do código solicitado pelo usuário.

## Tipos de Análise
1. **Arquitetura** - Estrutura geral, dependências, módulos
2. **Qualidade** - Code smells, duplicação, complexidade
3. **Performance** - Queries N+1, loops pesados, cache
4. **Segurança** - SQL injection, XSS, permissões
5. **Manutenibilidade** - Documentação, testes, padrões

## Método
- Usar Task tool com Explore agent para codebase
- Ler arquivos relevantes
- Buscar padrões com Grep
- Gerar relatório estruturado

## Output
- Pontos fortes
- Problemas identificados
- Recomendações priorizadas
- Plano de ação (se solicitado)
