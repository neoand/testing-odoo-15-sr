HR Attendance Reason - Employee Check In / Check Out Reason

This module helps the Employees to log their Check In / Check Out Reasons while using attendance check in / check out.

Features

- Create Check In / Check Out Reason on Backend.
- Select Reason from Check In / Check Out Reason.
- Create Dynamic Reason On the fly.
- Select Reason from Kiosk Mode.

Installation

- Copy attendance_reason module to addons folder

- Install the module normally like other modules

1. Create Reaosns

Attendances --> Configurations --> Reasons -->Attendance Reasons





2. Attendance Check In / Check Out

Attendances --> Check In / Check Out





3. Attendance Reason
